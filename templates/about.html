About Django

Django is an open-source web framework that makes common development tasks fast and easy. Because Django lets you take web applications from conception to launch within hours, your organization can be more productive.

Our History

In fall 2003, developers Adrian Holovaty and Simon Willison ditched PHP and began using Python to develop websites for World Online (the web department at the World Company in Lawrence, Kansas). As Holovaty and Willison built intensive, richly interactive sites such as Lawrence.com, they began to extract a generic web development framework that let them build web applications more quickly, following the best practices of web development. In summer 2005, World Online decided to open-source the resulting software, Django.

Django Software Foundation

Django is supported by the Django Software Foundation, an independent 501(c)(3) non-profit. We promote, support, and advance the open-source Django web framework by:

  • Fostering the development of Django by sponsoring sprints, meetups, gatherings, and community events.
  • Promoting the use of Django among the world-wide web development community.
  • Protecting the intellectual property and the framework's long-term viability.
  • Advancing state-of-the-art web development.
